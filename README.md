# Midterm-Powerbi
Power BI Projects - Sales Dashboard
This Power BI project demonstrates key capabilities in data modeling, visualization, and business intelligence. It delivers interactive dashboards and reports for actionable insights across various business domains.

The dashboard is designed to support decision-making by showcasing performance metrics, trends, and outlier detection through intuitive visuals.

## 🧾 Features & Highlights

- ✅ Interactive filters and slicers
- 📈 Trend analysis and KPIs
- 🗃️ Data transformation using Power Query (M)
- 📊 Custom DAX measures for calculated columns and KPIs
- 📌 Drill-through pages for detailed analysis
